# 檄文写作

檄文是"奉辞伐罪"之文：先证明自己有资格讨伐，再骂得有理有据有细节，展示实力震慑对方，最后给出路、下通牒。气势来自**结构与修辞的配合**，不是堆砌狠话。

本技能分三层：**第一层是本文件**，提炼自五篇历史名檄（陈琳讨曹操、祖君彦讨隋炀帝、骆宾王讨武曌、宋濂谕中原、曾国藩讨粤匪）的结构、修辞与套语，足以独立写出一篇合格檄文；**第二层是 `references/` 中的四份精讲笔记**（逐篇精读、骈句声律、用典骂法库、戏仿范例），要写出媲美名篇的作品必读其中相关篇目；**第三层是 `references/originals/` 中的五篇原文**，用于沉浸语感与核对引文。后两层缺失时照常写作，不要因缺少参考而降低标准。

## 第一步：判定模式与体式

**模式**（据讨伐对象判定，拿不准时问用户）：
- **严肃仿古**：对象是历史人物/事件，或用户明确要求庄重古文。全文文言，可用骈体，用典须准确。
- **现代戏仿**：对象是现代事物（烂代码、加班、涨价、室友熬夜……）。幽默引擎是**大词小用**——形式越庄重、对象越琐碎，反差越好笑。切忌中途破功掺入网络流行语或现代白话；一破功，喜剧张力即泄。现代专名（如"需求文档""外卖"）可保留，但要用文言句法包裹（"晨会未散，需求又至"）。

**体式**（据受众判定）：
- **檄**：写给敌人及天下看的，重数罪与威慑（陈琳、祖君彦、骆宾王体）。
- **谕**：写给要争取的民众看的，重大势与承诺（朱元璋体）。用户说"劝""告""动员"时多为谕体。

## 第二步：按五段骨架布局

历代名檄的共同骨架。短文（三百字内）可合并二三段、四五段，但**五个功能一个不能少**：

### 一、立论明义（开篇不骂人）
以天道、历史、公理起笔（"盖闻明主图危以制变""自古帝王临御天下"），先占领道德制高点。为什么：檄文的逻辑是"讨伐是义务而非私怨"，开篇立不住，后文的骂就成了泼妇骂街。卫道立论是高级打法——把对象塑造成整个文明秩序的敌人（曾国藩不说太平军反清，而说它毁灭"开辟以来之名教"），声讨抽象事物（制度、风气）时尤其好用。

### 二、数罪（全文重心，篇幅最大）
- **从出身骂起再骂罪行**：陈琳先骂曹操祖父是宦官、父亲买官（"赘阉遗丑"），骆宾王先骂武则天出身低微（"地实寒微"）。人格合法性先摧毁，罪行才显得"其来有自"。
- **罪状必须具体**：写到人名、事件、数字才有杀伤力。"残害忠良"是空话，"故九江太守边让……身首被枭悬之诛"是钉子。戏仿模式同理：骂屎山代码不要说"混乱不堪"，要说"一函数三千行，嵌套九重；变量名曰 data2、曰 temp_final_v3"。
- **多罪分条**：罪状超过五条时用祖君彦式分条（"其罪一也……其罪十也"），每条聚焦一个领域，条末点题。
- **收束用极致夸张**："罄南山之竹，书罪未穷；决东海之波，流恶难尽"——数罪之后必须有一句把罪恶推向无穷的总括。

### 三、扬我军威（述己方之义与之强）
自述正统与实力，夸张到宇宙级："以此制敌，何敌不摧；以此图功，何功不克""呼吸则河渭绝流，叱吒则嵩华自拔"。谕体在此段自述功业与合法性（"予本淮右布衣……今十有三年"），以经历代替威吓。

### 四、陈利害、开生路（分化敌营）
只骂不招降的檄文是失败的——目的是瓦解而非结仇。给出**具体的**出路与赏格：陈琳"得操首者封五千户侯，赏钱五千万"；曾国藩分五层招抚，每层各有待遇。戏仿模式把赏格换成等价物（"能重构此模块者，本部堂请以奶茶三月"）。

### 五、通牒收束（程式化结尾）
最后通牒（"若守迷不反，玉石俱焚，尔等噬脐，悔将何及"）+ 程式套语收束。经典结尾套语，任选其一或化用：
- 布告天下，咸使闻知
- 移檄州郡，咸使知闻
- 檄到如律令，无忽！
最强收法是骆宾王式反问："请看今日之域中，竟是谁家之天下！"——把结论抛给读者自己喊出来。

## 修辞武器库

- **骈四俪六**：四字句与六字句交替对仗，是气势的节拍器。不必全篇皆骈，关键段落（数罪总括、军威、结尾）务必用骈句推高潮。
- **用典**：以史为镜是檄文的论证方式（骂弑君比商臣，骂酗酒比殷辛）。严肃模式用典须准确；戏仿模式可将今事嵌入古典框架。
- **动物化贬称**：豺狼、枭獍、犬豕牛马、穷奇猰貐——对敌人不称其名而称其类。
- **天命符谶**（可选）：星象、童谣、谶纬证明"彼当亡我当兴"。戏仿模式可替换为现代征兆（"CI 十连红，此天亡之象也"）。
- **排比反问连击**：连续反问是情绪最高点的标配（"何城不陷？何阵不摧？"）。

## 套语与句式库（已内化，直接取用或化用）

以下句式采自五篇名檄，是檄文的"骨血"。化用优于照抄——替换其中的名物，保留其节奏。

**开篇立论**：
- 盖闻明主图危以制变，忠臣虑难以立权（陈琳式，"盖闻+公理"）
- 自古帝王临御天下，……未闻……者也（朱元璋式，"自古+正例，未闻+反例"，反例即讨伐对象所为）
- 为传檄事：逆贼××称乱以来，于今×年矣（曾国藩式，公文体直入，先摆祸害时长与规模数字）

**数罪句式**：
- 出身攻击：××者，性非和顺，地实寒微 / ××赘阉遗丑，本无懿德
- 分条点题：……。其罪一也。（每条先陈古之正道，再写其如何背道而驰，末四字点题）
- 罪状总括：罄南山之竹，书罪未穷；决东海之波，流恶难尽 / 人神之所同嫉，天地之所不容 / 历观载籍，无道之×，贪残酷烈，于××为甚

**扬威句式**：
- 排比反问：以此制敌，何敌不摧；以此图功，何功不克
- 宇宙级夸张：呼吸则河渭绝流，叱吒则嵩华自拔 / 喑呜则山岳崩颓，叱吒则风云变色
- 势如破竹：譬犹决沧海而濯残萤，举昆仑而压小卵 / 若举炎火以概飞蓬，覆沧海以沃熛炭，有何不灭者哉

**招降与通牒句式**：
- 赏格：其得××者，封五千户侯，赏钱五千万（数字务必具体）
- 分层招抚：倘有……者，本部堂……；倘有……者，……（曾国藩式，一层一待遇）
- 通牒：若守迷不反，昆冈纵火，玉石俱焚，尔等噬脐，悔将何及 / 归我者永安于××，背我者自窜于××

**结尾套语**（任选或化用）：布告天下，咸使闻知 / 移檄州郡，咸使知闻 / 檄到如律令，无忽！/ 最强收法是反问："请看今日之域中，竟是谁家之天下！"

**骈句节奏模板**：四字对四字（"班声动而北风起，剑气冲而南斗平"）、六字对六字（"一抔之土未干，六尺之孤何托"）、四六相间。自造骈句时先写上联，逐字位对仗写下联：名词对名词、动词对动词、数字对数字。

## 输出格式

- 标题：《讨××檄》《谕××檄》或《为××讨××檄》。
- 篇幅：默认四百至八百字（骆宾王全文仅三百余字而成千古名篇——宁短而精，勿长而散）；用户要求长文时可仿祖君彦铺陈至千字以上。
- 正文用段落分隔五段骨架，不加小标题（檄文是一气呵成的文章，不是报告）。
- 交稿后可附一句话说明写法（用了哪篇的路数），不要逐句注释。
