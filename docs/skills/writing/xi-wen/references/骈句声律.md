# 骈句声律：檄文气势的节拍器

骈句写不工整、读起来节奏不对时读本文。檄文不必全篇皆骈（陈琳、曾国藩以散体为主），但**数罪总括、扬威、收束**三处必须用骈句推高潮。

## 一、四种基本句式

以名檄原句为格：

| 句式 | 例句 | 出处 | 适用位置 |
|---|---|---|---|
| 四四对 | 铁骑成群，玉轴相接 / 杀姊屠兄，弑君鸩母 | 骆宾王 | 叙事、数罪，行文的基本步速 |
| 六六对 | 一抔之土未干，六尺之孤何托 / 罄南山之竹，书罪未穷；决东海之波，流恶难尽 | 骆宾王、祖君彦 | 煽情、总括，全篇警句多出于此 |
| 七字对（虚词作枢纽） | 班声动**而**北风起，剑气冲**而**南斗平 / 喑呜**则**山岳崩颓，叱吒**则**风云变色 | 骆宾王 | 扬威，动感最强 |
| 四六隔句对 | 身处三公之位，而行桀虏之态 | 陈琳 | 转折指控，"地位/行为"反差 |

## 二、对仗的工序（自造骈句四步法）

以"声讨外卖涨价"为例演示：

1. **定意**：这一联要说什么——"价日涨而量日减"。
2. **裁上联**：先把上联写顺，定下字数与词性骨架："餐价逐月而攀"（名词＋副词＋虚词＋动词）。
3. **对下联**：逐字位对仗——名词对名词、动词对动词、数字对数字、方位对方位："分量随日而缩"。
4. **调声**：朗读检查。大原则：**上联末字仄声（三四声），下联末字平声（一二声）**，读起来先抑后扬。不必严守四声八病，以朗读顺口、抑扬顿挫为准。

成品："餐价逐月而攀，分量随日而缩。"——可再升格加典："昔一金而饱腹，今再倍而犹饥。"

## 三、领字与虚词：骈文的关节

骈句连用易板滞，靠领字转气。常用领字按功能分类：

- **递进升级**：加以、况复、至乃、遂令、又
- **转折指控**：而（"圣人本意，惟避风雨；**而**不遵古典……"——正道与背离之间的那个"而"字是数罪的枢纽）
- **反问蓄势**：岂、讵、宁、焉能、乌可（"焉能展其节？""又乌可袖手安坐？"）
- **推论收束**：是以、于是、故、则
- **排比发端**：或……或……或……（"或家传汉爵，或地协周亲，或膺重寄于爪牙"）；倘有……倘有……（曾国藩五层招抚）

## 四、气势递进三连

情绪最高点的标准做法：**陈述 → 对偶 → 反问**，一浪高过一浪：

> 义旗所指，人心所向。（陈述）
> 班声动而北风起，剑气冲而南斗平。（对偶）
> 以此制敌，何敌不摧；以此图功，何功不克！（排比反问）

写到扬威段和收束段，检查是否有这个三连；没有就是平的。

## 五、数字与方位的对仗资源

檄文爱用数字、方位撑开空间感，对仗时同类相对：

- 方位铺张："南连百越，北尽三河"（骆）/ "西抵巴蜀，东连沧海，南控闽越"（朱）——写势力范围必用。
- 数字入罪："荼毒生灵百馀万，蹂躏州县五千馀里"（曾）/ "逆折十年之租"（祖）——数字是最硬的罪证。
- 数字对数字："封五千户侯，赏钱五千万"（陈）——赏格的数字要具体到可以兑现的程度。

## 六、常见毛病自查

- **一骈到底**：全篇四六，读者窒息。骈散相间，叙事用散、高潮用骈（学陈琳、曾国藩）。
- **对而不工**：词性错位（名词对了动词）。回到四步法逐字检查。
- **凑字虚胖**：为凑四六硬填"之""者"。删掉后意思不变的字都该删。
- **句脚同声**：连续几联下联末字全仄，读起来一路下坠。交替平仄收尾。
- **戏仿破功**：现代词裸露在骈句里（"KPI 逐月而攀"节奏即碎）。现代专名要么换译（KPI→考绩），要么放进散句缓冲，不入对仗字位。
