# 逐篇精读：五篇名檄的结构映射与技法拆解

写作前读本文对应篇目的小节，看大师如何执行五段骨架。需要全文语感时再读 `originals/` 下的原文。

---

## 一、陈琳《为袁绍檄豫州文》（讨曹操）——数罪与心理战的范本

**结构映射**：

| 原文段落 | 骨架功能 | 技法 |
|---|---|---|
| "盖闻明主图危以制变……" | 立论 | 公理起笔＋历史正例（赵高乱秦→望夷之败；产禄专政→绛侯朱虚兴兵），为"大臣讨伐当权者"预先正名 |
| "司空曹操祖父中常侍腾……" | 数罪·出身 | 三代连坐：祖父宦官（"并作妖孽"）→父亲买官（"舆金辇璧，输货权门"）→本人"赘阉遗丑，本无懿德" |
| "而操遂承资跋扈……" | 数罪·罪行 | 罪状全部落到实名实事：杀边让（"身首被枭悬之诛"）、迫害杨彪、杀赵彦、掘梁孝王墓（"破棺裸尸"）、设"发丘中郎将、摸金校尉" |
| "幕府奉汉威灵……" | 扬威 | 四面合围的空间铺陈（并州越太行、青州涉济漯、大军泛黄河、荆州下宛叶），再以比喻收（"举炎火以概飞蓬"） |
| "又操军吏士……咸怨旷思归" | 分化 | **心理战**：指出敌军士卒都是被裹挟的思乡之人，"扬素挥以启降路，必土崩瓦解，不俟血刃" |
| "其得操首者，封五千户侯……" | 赏格＋通牒 | 数字精确；"部曲偏裨将校诸吏降者，勿有所问"——首恶必办、胁从不问 |

**最值得偷师的三招**：
1. **恩义反衬**：数罪之前先历数袁绍对曹操的恩（授兵、表官、救其于吕布之难），"则幕府无德于兖土之民，而有大造于操也"——先施恩再控诉忘恩负义，讨伐立刻从私斗变成义举。
2. **官职与行为的反差句**："身处三公之位，而行桀虏之态，污国虐民，毒施人鬼"——地位越高罪越重，此句式可直接化用（"身居××之职，而行××之实"）。
3. **给敌人发明官职**："发丘中郎将、摸金校尉"是给盗墓行为编制官衔，讽刺力远超直骂。戏仿模式的利器：给讨伐对象的恶行"授官"（如给屎山代码设"祖传屎山守护中郎将"）。

## 二、祖君彦《为李密檄洛州文》（讨隋炀帝）——十罪铺陈的教科书

**结构映射**：立论（帝王当"驭朽索而同危"，未有暴虐克终者）→ 十罪 → 总括（"罄南山之竹"）→ 天命符谶 → 魏公家世与军威 → 战况与粮仓 → 招降 → 通牒 → "布告海内，咸使闻知"。

**十罪的领域划分**（每罪一个独立领域，绝不重复）：

| 罪 | 领域 | 钉子细节 |
|---|---|---|
| 一 | 弑父屠兄 | "先皇大渐……便行鸩毒"；比商臣、莒仆 |
| 二 | 乱伦无度 | 逼幸兰陵公主；"牝鸡鸣于诘旦" |
| 三 | 酗酒荒政 | "朝谒罕见其身，群臣希睹其面"；比殷辛、汉灵 |
| 四 | 大兴土木 | "穷生人之筋力，罄天下之资财"；比璿室、阿房 |
| 五 | 苛捐杂税 | "逆折十年之租"——预征十年，数字入罪 |
| 六 | 巡游筑城 | "尸骸蔽野，血流成河" |
| 七 | 穷兵黩武 | 征辽"只轮莫返"；比夫差黄池、苻坚寿春 |
| 八 | 拒谏杀贤 | 实名：高颎、贺若弼"暂吐良药之言，翻加属镂之赐" |
| 九 | 卖官鬻爵 | "钱神起论，铜臭为公" |
| 十 | 赏罚失信 | "危急则勋赏悬授，克定则丝纶不行"；比项王刓印 |

**每条罪的内部结构**（固定三步，务必学会）：先陈古之正道（"圣人本意，惟避风雨"）→ 再写其背道而驰（"而不遵古典……广立池台"）→ 末以四字点题（"其罪四也"）。正道立得越朴素，背离显得越荒唐。

**最值得偷师的三招**：
1. **数罪总括的双重无穷**："罄南山之竹，书罪未穷；决东海之波，流恶难尽"——竹写不完、水洗不净，两个维度的无穷。总括必须比任何单条罪状更夸张一级。
2. **天命段的证据链**：谶语（"隋氏三十六年而灭"）＋星象（"岁星入井"）＋异象（"朱雀门烧，正阳日蚀"）——三类证据排比，营造"天人共弃"。
3. **招降段的典故对**："审配死于袁氏，不如张郃归曹；范增困于项王，未若陈平从汉"——用两组"愚忠者死、改换者荣"的史例代替说教，劝降不言劝。

## 三、骆宾王《为徐敬业讨武曌檄》——短篇精悍与情感煽动的极致

**结构映射**（全文仅三段，五个功能压缩其中）：
- **第一段＝立论＋数罪**：首字定性"**伪**临朝武氏者"——一个"伪"字就是全篇立论。人格摧毁（"性非和顺，地实寒微"）→秽行（"入门见嫉……狐媚偏能惑主"）→大罪递进（"杀姊屠兄，弑君鸩母"，四字四罪，由亲及君）→总括（"人神之所同嫉，天地之所不容"）→现实危机（"君之爱子，幽之于别宫"）。
- **第二段＝举义＋扬威**：先立自家正统（"皇唐旧臣，公侯冢子"），再军容（"铁骑成群，玉轴相接"），后气势骈句连击（"班声动而北风起，剑气冲而南斗平……以此制敌，何敌不摧"）。
- **第三段＝策反＋通牒＋收束**：对象突然转向"**公等**"——直接向在朝旧臣喊话；情感杀招"**一抔之土未干，六尺之孤何托**"（先帝坟土未干、幼主无人可托，两个具象画面完成所有煽动）；利害两开（"共立勤王之勋……凡诸爵赏，同指山河"对"坐昧先几之兆，必贻后至之诛"）；反问收束"请看今日之域中，竟是谁家之天下！"

**最值得偷师的三招**：
1. **具象胜于抽象**：不说"先帝尸骨未寒而幼主危殆"，说"一抔之土未干，六尺之孤何托"——土是可见的，孤是可怜的。煽情必须给画面。
2. **第二人称的突袭**：前两段说"她"和"我"，第三段突然"公等"——读者被点名，无法再当旁观者。
3. **篇幅纪律**：三百余字成千古名篇。每段只做一件事，做完即走，不恋战。写短檄时以此篇为度。

## 四、宋濂《谕中原檄》（朱元璋讨元）——谕体与政治纲领的范本

**结构映射**（谕体：受众是要争取的民众，不是敌人）：
- 开篇点明受众："檄谕齐鲁河洛燕蓟秦晋**之人**曰"。
- 天命立论："自古帝王临御天下，中国居内以制夷狄……**未闻**以夷狄居中国治天下者也"——"自古……未闻……"句式，用全称历史否定对方存在的合法性。
- 数罪落在纲常：大德废长立幼、泰定以臣弑君、天历以弟鸩兄、"弟收兄妻，子烝父妾"——专挑礼崩乐坏的事，因为立论在纲常。
- 断其天命："古云：胡虏无百年之运，验之今日，信乎不谬！"——引谚语作结案。
- **排除法论证**：先历数河洛、关陕群雄皆"假元号以济私""志在养力"，"皆非华夏之主也"——把竞争者逐一排除，主角才出场。
- 自述以履历代替威吓："**予本淮右布衣**，因天下乱，为众所推……今十有三年"——低起点＋实绩，比任何夸张都可信。
- 八字纲领＋政策承诺："驱逐胡虏，恢复中华，立纲陈纪，救济斯民"；"兵至，民人勿避""归我者永安于中华，背我者自窜于塞外"。
- 宽宏收尾：蒙古、色目"愿为臣民者，与中夏之人抚养无异"——对最该恨的人网开一面，反而证明前文全部成立。

**最值得偷师的三招**：
1. **口号八字化**：政治纲领必须压缩到可传诵的程度，四组四字，动宾工整。谕体的核心交付物就是这句口号。
2. **排除法**：立自己之前先废他人，"当降生圣人"的期待落空（"今壹纪于兹，未闻有济世安民者"）制造真空，主角登场即是必然。
3. **给受众行为指引**：谕体必须告诉读者"你该做什么"——"兵至，民人勿避"六个字就是全部动员。

## 五、曾国藩《讨粤匪檄》——卫道立论与分层招抚的范本

**结构映射**：
- 开篇公文体："为传檄事：逆贼洪秀全、杨秀清称乱以来，于今五年矣。荼毒生灵**百馀万**，蹂躏州县**五千馀里**"——不引经典，直接上数字。
- 罪状白描（不用典，用细节）："银满五两而不献贼者，即行斩首""男子日给米一合""妇女不肯解脚者，则立斩其足以示众妇"——近乎新闻记录的残酷细节，比任何修辞都重。
- **三重愤怒递进**（全篇脊柱）：①人之愤——荼毒生灵；②圣贤之愤——毁孔孟名教，"此岂独我大清之变，乃开辟以来名教之奇变，我孔子、孟子之所痛哭于九原"；③鬼神之愤——焚学宫、毁关帝岳王庙，"斯又鬼神所并愤怒"。讨伐层级从朝廷之敌升格为文明之敌、天地之敌。
- **对比杀招**：李自成至曲阜不犯圣庙，张献忠至梓潼亦祭文昌——连流寇都敬圣人，粤匪连流寇都不如。
- **五层招抚**（一层一待遇，精确对价）：号召义旅者→"引为心腹，酌给口粮"；抱道君子→"礼之幕府，待以宾师"；捐银助饷者→"千金以内给予实收部照，千金以上专折奏请优叙"；杀头目以城来降者→"收之帐下，奏授官爵"；被胁弃械归诚者→"一概免死，资遣回籍"。
- 通牒＋赌咒收束："大兵一压，玉石俱焚"；"上有日月，下有鬼神……实鉴吾心。咸听吾言！檄到如律令，无忽！"

**最值得偷师的三招**：
1. **卫道立论**：把对象塑造成"开辟以来名教之奇变"，敌人就不再是某个政权的敌人，而是所有读书人、所有信神者的敌人——同盟最大化。讨伐抽象事物（制度、风气）时，这是唯一正确的立论法。
2. **细节即罪证**：白描的残酷细节（斩足示众、倒抬其尸）无需形容词。收集讨伐对象的"现场细节"永远优先于堆砌骂词。
3. **招抚要给对价表**：五层人群、五种待遇、金额分档——像商业条款一样精确。模糊的"来降不杀"没有说服力。

## 横向总结：五篇的分工

| 你要写的东西缺什么 | 学谁 |
|---|---|
| 罪状不够具体、骂不到痛处 | 陈琳（实名实事）、曾国藩（白描细节） |
| 罪多而乱、不成体系 | 祖君彦（分领域、其罪N也、三步内构） |
| 篇幅冗长、缺一击必杀的句子 | 骆宾王（具象煽情、反问收束、三段纪律） |
| 立论软弱、师出无名 | 朱元璋（自古未闻＋排除法）、曾国藩（卫道升格） |
| 结尾泄气、招降空洞 | 陈琳（精确赏格）、曾国藩（分层对价） |
