# 戏仿范例：《讨会议檄》全文与拆解

写现代戏仿檄文前读本文，看"大词小用"如何落地。范例之后附逐段拆解与戏仿三律。

## 范例全文

> **《讨会议檄》**
>
> 盖闻上古结绳而治，事简而功成；圣人垂拱无为，言寡而天下理。未闻聚众终日、议而不决，可以成业者也。
>
> 会议者，窃"协作"之美名，行耗时之实祸。晨光初启，日历已满；一议未散，一催又至。发起无端，议程不设，其罪一也。十人环坐，一人独言，言者滔滔若江河之决，听者昏昏如泥塑之陈，其罪二也。幻灯百页，要义无一，"对齐"再三，愈对愈歧，其罪三也。及其散也，结论杳然，行动无主，惟曰"下次再议"，其罪四也。遂令代码无人写，文档无人修，工期日蹙而会期日增。罄打工人之肝，加不完之班；决咖啡机之流，提不起之神。历观载籍，误事之术，繁苛酷烈，于会议为甚！
>
> 今有司渐悟，同侪思变。凡我同列，孰无案牍之劳？孰无交付之期？众怒所指，如火燎原。以此拒会，何会不辞；以此专注，何功不成！
>
> 倘有先设议程、限时半刻者，吾辈欣然赴之；倘有以一邮代一会者，尊为上宾；倘有径裁例会者，谢以奶茶三月。若夫积习不改，滥发邀约，则众必静音以对，镜头长闭，应者寥寥，主持之人独对虚室，悔将何及！
>
> 移檄诸群，咸使闻知。

## 逐段拆解

| 段 | 骨架功能 | 用了什么招 |
|---|---|---|
| 盖闻上古结绳…… | 立论 | "盖闻＋公理"开篇（陈琳式）；"未闻……者也"全称否定（朱元璋式）。立论选"简胜于繁"，为全篇定理据 |
| 会议者……于会议为甚 | 数罪 | 先揭其"窃美名行实祸"（曾国藩卫道式定性）；四罪分条（祖君彦式，每条一个领域：滥发、低效、空转、无果）；细节具体（幻灯百页、下次再议）；总括化用"罄……决……"双无穷＋"历观载籍，于××为甚"（陈琳） |
| 今有司渐悟…… | 扬威 | 气势递进三连：陈述→反问排比（"以此拒会，何会不辞"化用祖君彦） |
| 倘有……悔将何及 | 招抚＋通牒 | 曾国藩五层招抚缩为三层，一层一对价（奶茶三月＝赏钱五千万的等价物）；通牒给出具体后果画面（独对虚室） |
| 移檄诸群，咸使闻知 | 收束 | 程式套语，"州郡"换"诸群"——古今词替换恰好构成最后一个笑点 |

## 戏仿三律

1. **形式全程庄重，笑点全部外包给反差**。文中没有一个网络梗、没有一处自我调侃；幽默完全来自用讨伐隋炀帝的规格声讨开会。一旦忍不住插一句"绝绝子"，反差塌方，全篇变成普通段子。
2. **现代名物要么换译、要么散句缓冲**。"幻灯百页"（PPT→幻灯）入了骈句；"奶茶""静音""镜头"保留原词但放在散句里，不占对仗字位（参见 骈句声律.md 第六节）。判断标准：朗读时节奏不断即可留。
3. **罪状仍须真实具体**。戏仿的说服力和严肃檄文同源——细节。"十人环坐，一人独言"人人见过，才会会心；虚构的夸张罪状（"一会开三日"）反而弱，因为失去了"被说中"的快感。夸张只用在总括（"罄打工人之肝"），不用在罪状本身。

## 常见翻车点

- **通篇堆砌"之乎者也"而无骈句骨架**——古文感不来自虚词密度，来自对仗与节奏。
- **骂得太狠失了分寸**——戏仿对象常是同事、室友、公司，收束要留温度（本篇通牒的后果只是"独对虚室"，不是"玉石俱焚"）。招抚段给台阶，正是檄文体自带的分寸。
- **忘了自己是谁**——檄文必有讨伐主体。本篇是"吾辈同侪"；写《谕熬夜檄》主体可以是"本舍长"。主体立住，"本部堂""吾辈"等自称才有喜剧威严。
