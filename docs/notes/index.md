# 笔记

平时看到的小问题、想通的模型细节，记在这里。和[文章](../posts/index.md)的长文叙事、[工作口味](../skills/index.md)的成套实践不同，这里是原子化的小笔记：每篇以一个问题开头，一题一悟。

每篇的骨架统一是：**题 → 一句话 → 关键技巧 → 解 → 延伸**。价值密度在"关键技巧"——记的不是答案，是可迁移的招；分类索引里每篇只留一行钩子，扫一眼决定点不点进去。
